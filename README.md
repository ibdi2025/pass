# pass
Cielom projektu 'Pass' je Python program na vytvaranie uzivatelskych uctov a ich zapisovanie do suboru 'pass.txt'.

Program zapise do suboru uzivatelske meno a hash hesla.

V pripade, ze uzivatelske meno uz existuje, program sa ukonci.

Subor bude nasledne pouzity na simulaciu prihlasenia sa do systemu overenim hashu hesla ulozeneho v subore 'passwd.txt'.
